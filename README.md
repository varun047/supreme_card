# Project Name
### supreme_card

A Python-based implementation of a card game.

# About
 supreme_card is an open-source Python project designed to simulate a basic card game. The project utilizes object-oriented programming principles to provide a structured approach to managing cards, players, and game logic.

## Tech Stack
- **Programming Language:** Python 3.x
- **File Structure:**
  - `deck.py`: Handles card creation and management.
  - `distribution.py`: Manages the distribution of cards among players.
  - `p.py`: Provides a player class for tracking player actions and status.
  - `players.py`: Handles player interactions and game state.

# Features
- Supports multiple players
- Card distribution based on pre-defined deck
- Simple game logic implementation

# Getting Started
1. Clone the repository using `git clone` command.
2. Navigate into the project directory with `cd`.
3. Install dependencies by running `pip install -r requirements.txt`.
4. Run the game with `python main.py`.
